## 标准用户
#### 标准用户界面和管理员公用一个界面，只有对博客的编辑权限，但没有对用户的增删修改的权限。页面也隐藏到了相关按钮。
## 未登录状体
#### 未登录状态下只能查看，无法编辑和修改。界面也和管理员相同，只不过所有权限按钮被隐藏了。
